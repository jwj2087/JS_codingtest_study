2. 새롭게 알게 된 점
    - Math.trunc() : 소수점 이하 버리기
    
    ```jsx
    // Math.trunc()와 Math.floor() 비교
    console.log(Math.trunc(-23.3)); // -23, 소수점 이하 버리기
    console.log(Math.floor(-23.3)); // -24, 소수점 내림
    ```
    
    - Math.ceil() : 반올림
    - String도 length 프로퍼티 사용 가능
